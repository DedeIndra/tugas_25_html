Mengenal Semantic Elements Di HTML5

  • article

    Ketika kita ingin membuat layout untuk menampilkan blcok artikel-artikel yang ada di website kita , tag article bisa kita gunakan dengan element header
    karena tag article memiliki type blcok. Artinya element ini memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya.

  • Aside

    Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah po
    karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
